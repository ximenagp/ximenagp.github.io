Proyecto Webmapping

Visualizacion de informacion geografica en la web

Descripcion del proyecto

Este proyecto consiste en la creacion de un webmapping orientado a la visualizacion de datos geograficos mediante mapas interactivos. Su objetivo es facilitar la exploracion de informacion espacial de forma clara, dinamica y accesible para todos los usuarios.

A traves de este desarrollo, se integran herramientas web para representar ubicaciones, capas de informacion y elementos visuales que permiten acceso virtual a informacion publica, que el usuario acostumbra a consultar en un lugar fisico.
